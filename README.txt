OpenClassroom - Cours Git-GitHub - Partie 2 - Activité

Ce projet est un programme informatique contenant un menu qui permet d'afficher une blague ou une devinette.

Il est lançable depuis le terminal (invite de commande) en se positionnant dans le dossier et en tapant "./Programme", le programme se lancera dans votre terminal. 



Le dossier contient 5 fichiers:

-Le makefile, qui permet de compiler le code

-Le fichier 'main.cpp' qui contient le programme principal

-le fichier 'main.o' généré par le makefile

-l'éxécutable 'Programme'

-Le README.txt que vous êtes en train de lire !


Ce projet contient 6 commits.
